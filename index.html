Welcome to DMUPB project.

This project is developed by Jose Joya and Christian Velandia, students Universidad Pontificia Bolivariana / Bucaramanga, for the subjects of:

  • Software Engineering II
  • web development
  • Analysis and Design of Algorithms
  • telematics
  • Operating Systems

Project Objectives

General:

  • Develop a web tool engine MySql database and PHP programming language, basing their functionality in data mining to identify trends in groups of patients with symptoms of heart disease which can offer states or special situations.

Specifics:

  • Take requirements to establish the scope of the problem.
  • Apply standards and methodologies in the stages of analysis and design software.
  • Modeling and design requirements in the UML modeling language, using use case diagrams, class diagram, sequence diagram and activity diagram.
  • Design and implement the model Entity-Relationship (ER) and the build script database engine for MySQL databases.
  • Design and develop algorithms for analysis, sorting and grouping of the information in the database based on the data clustering technique Cluster analysis.
  • Applying the concept of multitasking processes for applying a resource management technique.
  • Understand and manage standard WAN technologies and protocols to maintain security between communications.
